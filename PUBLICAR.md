# Vigilia Seguridad Soporte Operativo

Esta carpeta ya esta lista para publicarse como web estatica.

Archivos necesarios:
- `index.html`
- `styles.css`
- `app.js`
- `data.js`

## Opcion rapida: Netlify

1. Entrar a https://app.netlify.com/drop
2. Arrastrar esta carpeta completa.
3. Netlify devuelve un link publico.
4. Cambiar el nombre del sitio desde Site settings si queres una URL mas prolija.

## Opcion ordenada: GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir estos cuatro archivos a la raiz del repositorio.
3. Ir a Settings > Pages.
4. Elegir Deploy from branch y seleccionar `main`.
5. GitHub genera una URL publica.

## Importante

No subir datos sensibles reales de clientes, telefonos, palabras clave, direcciones privadas o historiales internos sin autorizacion de la empresa.
